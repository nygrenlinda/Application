Explaining the term "Framework"

What is framework?

A framework serves as a basis for the organization and development of software. Think of it as a kind of template. For example, frameworks can consist of specific software artifacts and modules that helps structure and organize the process of a project.

What is it good for?

Using a a framework allows the work to be done easier and more effective. Working with frameworks also prevent possible risks of errors and makes it possible to achieve great results in minimum time.

Also..

Besides form making the work easier and effective, frameworks aslo offers special tools and open up for collaborations between coders. Angular JS, React JS, Vue JS and Bootstrap are examples of frameworks for Front-end programmers.
